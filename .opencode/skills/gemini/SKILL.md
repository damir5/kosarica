## What I do
Provide consultation with Google's Gemini CLI, an open-source AI agent that brings Gemini's power directly into your terminal for coding assistance, codebase analysis, and development workflows.

## When to use me
Use this skill when you need assistance from Gemini CLI for:
- Code generation and completion
- Codebase exploration and understanding
- Bug fixing and debugging
- Test generation
- Documentation creation
- Complex development tasks

## How to consult Gemini CLI
1. Install Gemini CLI: `npm install -g @google/gemini-cli`
2. Set up your API key: Configure Gemini API access
3. Run queries: `gemini "your coding task or question"`
4. For specific operations, use available commands and extensions

## Examples
- `gemini "implement a REST API endpoint"`
- `gemini "fix this bug in my React component"`
- `gemini "generate unit tests for this function"`

## Features
- Supports multiple Gemini models (including Gemini 3 Flash)
- Integrates with MCP servers for extended capabilities
- Can work with local codebases and remote repositories
- Supports various development workflows

## Best practices
- Provide clear, specific prompts for best results
- Include file paths or code snippets when relevant
- Use Gemini 3 Flash for faster responses on coding tasks
- Leverage MCP servers for additional tools and integrations
